WUWA The Smart Door App

WUWA is a voice-activated smart door system, that uses Voice Recognition to authenticate users before executing commands (open, close, lock) with Speech-To-Text.

The system leverages a Raspberry Pi for real-time edge processing and motor control, converting authenticated voice commands (via STT) directly into mechanical outputs to physically manipulate the mechanism.

The end Goal is to achieve a Keyless automated real sized door, with cool and stylized way to control the secured entrance.

A dedicated mobile application is In Development to manage user credentials add or remove users using a secure QR code, to dynamically train and add their voice models to the Raspberry Pi unit via the Flask Backend.

Skills: Voice Recognition, Speech-To-Text, PyQT, Raspberry, Embedded System, Edge Device, Mobile App, Flask, GUI
