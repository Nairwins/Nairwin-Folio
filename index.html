Black-Hole Simulation

Explored how light bends around one of the largest black holes known, Ton618, using C++ simulations with three approaches: 2D ray tracing, 3D OpenGL rendering, and a fully realistic visualization.

2D Ray Tracing: Minimal 2D simulation using the Schwarzschild formula to visualize how rays curve near the black hole’s center.

3D OpenGL Simulation: Extended into 3D with CUDA acceleration for heavy computations, allowing real-time visualization of bending rays in space.

Ton618 Realistic Visualization: Added spinning accretion disks, gravitational lensing, and Doppler color shifts for a lifelike black hole simulation.

This project was a fun way to combine physics, GPU computing, and graphics programming. Watching light bend in 3D space was truly amazing.

Skills: C++, CUDA, OpenGL, GLSL/Shaders, Ray Tracing, General Relativity, Parallel Computing, Physical Simulation, Performance Optimization
